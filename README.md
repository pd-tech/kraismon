# Kraismon
Krajta Interactive System monitor

## Requirments
* Python 3.5+ (tested on 3.6)
  * **asyncio** module
  * **psutil** module `pip install psutil`
* Internet connection
* OS: Win NT/Linux/macOS/BSD/Solaris

## User Guide
1. Launch monitoring daemon:
  `python3 kraismon.py`
2. Open `kraismon.html` in any modern web browser
